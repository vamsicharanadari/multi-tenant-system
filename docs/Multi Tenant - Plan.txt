Run command: cd backend && pytest tests/test_auth_and_context.py $\rightarrow$ Assert pass (GREEN).
BLOCK 3: Multi-Tenant Data Isolation (12:00 - 18:00)
Step 3.1 — Red Phase: Write Failing Tests

test_tenant_a_data_isolated_from_tenant_b(): Query database as Tenant A $\rightarrow$ verify zero visibility into Tenant B records.
test_cross_tenant_lookup_denied(): Request Tenant B resource ID with Tenant A JWT $\rightarrow$ assert HTTP 403 or 404.
Run command: cd backend && pytest tests/test_tenant_isolation.py $\rightarrow$ Assert failure (RED).
Step 3.2 — Green Phase: Implement Minimal Production Code
Write tenant database base schemas in backend/app/models/tenant.py (User model scoped to tenant DBs).
Configure dynamic session context resolution in API dependency wrappers (get_tenant_db).
Run command: cd backend && pytest tests/test_tenant_isolation.py $\rightarrow$ Assert pass (GREEN).
